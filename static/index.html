google-drive-ocamlfuse

FUSE filesystem for Google Drive

google-drive-ocamlfuse is a FUSE-based file system backed by Google Drive, written in OCaml. The project is hosted on github, where you can find the latest development version, and on the OCaml forge where you can find releases.

There is a mailing list where you can post question, suggestion or report problems. Or you may want to open an issue on github.
